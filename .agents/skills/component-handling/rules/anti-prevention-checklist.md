# anti-prevention-checklist

**Category:** Anti-Patterns
**Impact:** HIGH

## Rule

Before creating or moving a component, verify this checklist.

## Checklist

- [ ] Truly reusable or just used here?
- [ ] Importing in correct direction?
- [ ] Right level of complexity?
- [ ] Using existing molecules?
- [ ] Correct folder for purpose?
- [ ] Has tests (if molecule/organism)?
- [ ] Has Storybook (if molecule/organism)?
- [ ] Properly composed vs over-configured?
