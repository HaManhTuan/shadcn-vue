# Search

**Examples:** `SearchBar`, `SearchInput`

## What type of component is this?

Molecule

## Common use purpose

Input with search icon and clear button. Lets user type a query to filter lists or content. Used wherever text search/filter is needed.

## Naming recommendation

`SearchBar` or `SearchInput` — descriptive of the pattern (icon + input + optional clear)

## Where to place it

`src/components/molecules/search-bar/`

## Can it be used in multiple places?

Yes. Users list, products list, orders list — any feature that needs search/filter. Reusable across the app.
